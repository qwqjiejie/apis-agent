# 业务主流程

## 流程全景

```
浏览器 POST /api/v1/chat {"message":"生成ppt: Spring Boot介绍","conversationId":"sess_xxx"}
    │
    ▼
┌─ main.py ─────────────────────────────────────────────────────────────┐
│ 1. lifespan() 启动时创建 Agent 单例 → app.state.agent                  │
│ 2. log_requests 中间件注入 trace_id                                     │
│ 3. RateLimitMiddleware 限流检查                                        │
└───────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─ routes/agent.py:69 agent_chat() ─────────────────────────────────────┐
│ 4. 解析 ChatRequest → query, conversationId                           │
│ 5. 获取单例: agent = request.app.state.agent                           │
│ 6. 构造 config = {thread_id, recursion_limit}                         │
│ 7. 调用 agent.astream_events() 流式输出                                │
│ 8. 保存会话 _save_session()                                           │
│ 9. 生成标题 _generate_title()                                         │
└───────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─ deepagents 框架 (CompiledStateGraph) ────────────────────────────────┐
│ 10. SubAgentMiddleware 注入 task 工具                                  │
│ 11. LLM 判断 → 调用 task(name="ppt_specialist", ...)                  │
│ 12. 框架 spawn ppt_specialist 子代理（独立 tools + system_prompt）      │
│ 13. 子代理执行 → 结果返回主 Agent                                       │
│ 14. 主 Agent 输出给用户                                                 │
└───────────────────────────────────────────────────────────────────────┘
```
